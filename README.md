# BASIC-WEATHER-APP

## Description
A simple Weather App built using Python and Tkinter. It fetches real-time weather information for a city using the Open-Meteo API.

## Features
- Search weather by city
- Current temperature, humidity, and wind speed
- 6-hour forecast
- 5-day forecast
- Error handling

## Technologies
- Python
- Tkinter
- Requests
- Open-Meteo API
